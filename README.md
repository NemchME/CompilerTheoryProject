# CompilerTheoryProject

Проект реализует лексический и синтаксический анализатор подмножества языка Pascal.  
В результате разбора программы строится **абстрактное синтаксическое дерево (AST)**.

AST используется как промежуточное представление программы и подготовлено для
дальнейших этапов компиляции (например, семантический анализ).

---

# Поддерживаемые конструкции языка

## Объявления

Поддерживается секция объявления переменных:

```
var
  x: integer;
```

Поддерживаемые типы:

- `integer`
- `char`
- `boolean`

---

## Операторы

Поддерживаются следующие операторы.

### Присваивание

```
x := expression
```

### Условный оператор

```
if condition then
  statement
else
  statement
```

### Цикл while

```
while condition do
  statement
```

### Цикл for

```
for i := a to b do
  statement
```

или

```
for i := a downto b do
  statement
```

### Управление циклом

```
break
continue
```

### Ввод / вывод

```
read(x)
readln(x)

write(x)
writeln(x)
```

# Абстрактное синтаксическое дерево (AST)

После разбора программы строится AST.

В AST используются следующие узлы:

- `Program`
- `Block`
- `VarDecl`
- `CompoundStmt`
- `Assign`
- `If`
- `While`
- `For`
- `Break`
- `Continue`
- `Read`
- `Write`
- `BinOp`
- `UnOp`
- `Ident`
- `Literal`

Операции представлены перечислениями:

- `BinaryOpKind`
- `UnaryOpKind`

Это позволяет избежать строковых операторов и упрощает дальнейший анализ программы.

---

# Пример AST

Для программы:

```
program Demo;
var
  x: integer;
begin
  x := 2 + 3 * 4;
end.
```

строится дерево:

```
Program Demo
└─Block
  ├─VarDecl x : integer
  └─CompoundStmt
    └─Assign
      ├─Ident x
      └─BinOp +
        ├─Literal 2
        └─BinOp *
          ├─Literal 3
          └─Literal 4
```
